Default Kick Sounds

Place your default kick sound files here (WAV, MP3, or OGG format).
Recommended names:
- kick1.wav
- kick2.wav
- kick3.wav

You can find free kick sound samples from:
- https://freesound.org/
- https://www.zapsplat.com/
- https://www.soundjay.com/

After adding sound files, uncomment the corresponding lines in MainActivity.kt:
    // audioManager.addSoundFromResource("Kick 1", R.raw.kick1)
    // audioManager.addSoundFromResource("Kick 2", R.raw.kick2)
    // audioManager.addSoundFromResource("Kick 3", R.raw.kick3)
